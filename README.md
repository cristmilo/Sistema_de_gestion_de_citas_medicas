# Sistema de Gestión de Citas Médicas

## Descripción del proyecto

Este programa es un sistema básico de gestión hospitalaria desarrollado en Python utilizando Programación Orientada a Objetos (POO) y la biblioteca gráfica Tkinter.

El sistema permite registrar pacientes, registrar médicos, agendar citas médicas y visualizar las citas registradas mediante una interfaz gráfica organizada en pestañas.

---

## Requisitos del sistema

Para ejecutar este programa se necesita:

- Python 3 instalado
- Biblioteca Tkinter (normalmente ya viene incluida con Python)

---

## Instalación

1. Descargar o clonar el proyecto en su computador.
2. Guardar el archivo del programa (por ejemplo `Sistema_de_gestion_de_citas_medicas.py`) en una carpeta.
3. Abrir una terminal o consola en esa carpeta.

---

## Ejecución del programa

En la terminal ejecutar el siguiente comando:

python Sistema_de_gestion_de_citas_medicas.py

Esto abrirá la interfaz gráfica del sistema hospitalario.

---

## Uso del sistema

El programa contiene cuatro pestañas principales:

### 1. Registrar Paciente
Permite ingresar:
- ID del paciente
- Nombre
- Apellido
- Teléfono

Al presionar el botón **Guardar**, el paciente se registra en el sistema.

### 2. Registrar Médico
Permite ingresar:
- ID del médico
- Nombre
- Especialidad

El médico queda almacenado en el sistema.

### 3. Agendar Cita
Permite seleccionar un paciente, un médico y una fecha para registrar una cita médica.

### 4. Mostrar Citas
Al presionar el botón **Mostrar Citas**, el sistema muestra todas las citas registradas.
